# AWS環境構築手順

## 1. VPC作成
- サブネット：1つ
- ルートテーブル：インターネットゲートウェイを接続
- CIDR：172.31.0.0/16（デフォルト可）

## 2. セキュリティグループ設定
- インバウンド：22(SSH), 80(HTTP) を開放
- アウトバウンド：全許可

## 3. EC2インスタンス作成
- OS：Amazon Linux 2（無料枠対象）
- タイプ：t3.micro
- キーペア作成（.pemを安全に保管）

## 4. SSH接続
```bash
chmod 400 mykey.pem
ssh -i mykey.pem ec2-user@<パブリックIP>
